# 2D 횡스크롤 액션 게임

Unity로 제작한 간단한 2D 횡스크롤 액션 게임입니다. 캐릭터 이동, 점프, 연속 공격, 피격, 사망 애니메이션, UI 및 키보드 입력 기능이 포함되어 있습니다.

## 주요 기능

- 캐릭터 좌우 이동
- 점프 (물리 기반)
- 3단 콤보 공격
- 피격 및 사망 애니메이션
- 키보드 입력 제어
- ESC 키로 게임 종료
- 간단한 게임 UI 구현

## 조작 키

| 키   | 동작            |
|------|-----------------|
| ← →  | 좌우 이동        |
| Z    | 공격             |
| X    | 피격 애니메이션  |
| C    | 사망 애니메이션  |
| 스페이스바 | 점프     |
| ESC  | 게임 종료        |
